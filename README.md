# Portfolio Website

A personal portfolio website built with **React.js** to showcase projects, skills, and contact information.

---

## Features

* **Project Showcase:** Display your projects with descriptions and live demo links.
* **Skills Section:** Highlight your technical skills.
* **Responsive Design:** Works on desktop and mobile devices.
* **Contact Form:** Visitors can reach out directly through the website.

---

## Tech Stack

* **Frontend:** React.js, HTML, CSS, JavaScript
* **Design:** CSS Flexbox, Grid, and responsive techniques

## How to Run

1. Clone the repository:

   ```bash
   git clone <your-repo-link>
   ```
2. Navigate to the project folder:

   ```bash
   cd Portfolio
   ```
3. Install dependencies:

   ```bash
   npm install
   ```
4. Run the development server:

   ```bash
   npm run dev
   ```
5. Open [http://localhost:3000](http://localhost:3000) in your browser.
